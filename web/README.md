# Latero Insights Web Module

This directory contains the application module of Latero Insights.

Main contents:

- `src/` — Next.js application code
- `public/` — static assets
- `data/` — local writable app data
- `scripts/` — app-local scripts
- `.env.example` — web module environment template

Typical use from the repository root:

```bash
npm install
cp web/.env.example web/.env.local
npm run dev
```
